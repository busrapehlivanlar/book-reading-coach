# Book Reading Coach / Okuya

Book Reading Coach, kitap okumaya başlamakta zorlanan kullanıcılar için geliştirilmiş bir kişisel okuma alışkanlığı koçu prototipidir.

Kullanıcı 4 kısa soruya cevap verir; uygulama kullanıcının okur profilini, günlük okuma hedefini, başlangıç kitap önerilerini ve 7 günlük mini okuma planını gösterir.

## Canlı Demo

https://book-reading-coach.vercel.app

## Özellikler

- Kişisel okur profili testi
- Kullanıcının cevaplarına göre profil belirleme
- Günlük mikro okuma hedefi
- 7 günlük başlangıç planı
- Kitap öneri kartları
- Responsive landing page tasarımı
- Backend gerektirmeyen demo akışı

## Kullanılan Teknolojiler

- Next.js
- React
- TypeScript
- Tailwind CSS
- Vercel
